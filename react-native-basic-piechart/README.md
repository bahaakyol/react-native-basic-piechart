# react-native-basic-piechart

Pie Chart component for React Native.

## Installation

```sh
npm install react-native-basic-piechart
```

## Usage

```js
import PieChart from 'react-native-basic-piechart';

// ...

<View>
  <PieChart
  dataValues = ['20', '60', '70', '46', '30']
/>
</View>
```
